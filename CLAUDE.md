## Typography

Use a clean system-oriented stack. Prefer:
- `-apple-system`
- `BlinkMacSystemFont`
- `Inter`
- `SF Pro Display` / `SF Pro Text` if available on the device
- `system-ui`
- sensible fallbacks

### Typography rules
- The name should be large, bold, and calm, not flashy.
- The headline should be very readable and confident.
- Body text should be concise with high readability.
- Section titles should feel like product UI section headers, not blog headers.
- Avoid long paragraph walls.
- Use sentence case, not all caps for most content.
- Letter spacing should be subtle and controlled.
- Use stronger weight changes more than too many different font sizes.

### Suggested hierarchy
- Hero name: very strong, premium, simple
- Main headline: bold, slightly smaller than the name
- Supporting text: medium contrast, max width controlled
- Section labels: compact and refined
- Card titles: semibold
- Meta labels: smaller, quieter, but still readable

## Motion system

Motion is a major part of the premium feel.

Use motion that feels:
- springy
- soft
- physically believable
- never distracting

Good motion patterns:
- fade + upward settle
- soft scale on reveal
- parallax only if extremely subtle
- hover elevation
- dock-like hover response for icon rows if tasteful
- smooth section transitions
- content reveal stagger with restraint

Avoid:
- long dramatic animation sequences
- scroll-jacking
- huge elastic effects
- flashy rotating gradients
- constant motion in the background
- animation for everything

Motion should support clarity and quality, not show off.
